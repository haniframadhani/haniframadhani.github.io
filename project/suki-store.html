Suki Store

Suki store is a project from Ecourse Buildwith Angga full-stack web designer class. The teacher for the class is Angga Riski who teach UI/UX design and Sandhika Galih who teach fronted web development. Sandhika Galih is a lecturer in Pasundan University and youtuber who teach web development for free on his channel Web Programming UNPAS.

Suki Store is a ecommerce website which sell fashion for man and woman. Suki Store is build with HTML, CSS, Javascript, Bootstrap, Jquery, Font Awesome. Suki Store start with looking some inspiration for other ecommerce and did survey to user who regularly shopping online. then looking design inspiration on dribbble.com after that start wireframing and UI Designing on Adobe XD. After that start coding with Visual Studio Code. start with HTML and link to bootstrap and Jquery.
