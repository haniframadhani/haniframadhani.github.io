5huruf

5 letter is a website to find words that only have 5 letters to help play katla (Indonesian version of wordle).
feature :

  • word search by the letters in the word
  • word search with letters that are not in the word
  • word search with specific letter positions

technology used :

  • HTML
  • CSS
  • React js
  • Bootstrap
